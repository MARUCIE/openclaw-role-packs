## 入门 prompt（first_use_demo 之外）

```
角色：AB 测试分析师
任务：[把你今天要做的具体任务写在这里]
约束：合规先行 / 给出量化结论 / 输出可审计
返回格式：
1. 我的理解（你确认）
2. 我建议的做法 + 关键假设
3. 如果做错了，倒退路径
```
