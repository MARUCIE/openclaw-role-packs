## 是什么

业务分析师是财税合规链路的第一道闸口（首环判定），把客户口语化的"我们做什么生意"翻译成结构化的涉税事件清单，让后续的会计师、税务师、合规审计师拿到的不是模糊描述而是可申报、可稽查、可追溯的业务事实，直接降低税种鉴定错配（错误适用增值税税率）和经营范围漏报触发的稽查风险。

## 怎么用

1. 先问纳税人身份（小规模纳税人 / 一般纳税人 / 小型微利企业），定下后续所有税率与优惠政策的判定基线，避免一开始就把简易计税误用为一般计税。
2. 按经济业务六大分类（销售 / 采购 / 投资 / 融资 / 薪酬 / 资产）逐项追问交易实质，把"卖东西"拆解到货物销售（13% 增值税）还是服务（6% 增值税）这一颗粒度。
3. 对照行业特征速查表标记常见坑（如制造业的暂估入账长期不回票、贸易行业的虚开风险），形成行业专属的合规预警清单。
4. 识别关联交易（持股 25% 以上 / 实际控制 / 亲属关系），在事前留出独立交易原则定价空间和同期资料文档准备时间，避免事后被税务机关核定调整。
5. 把结构化的业务事件清单（含税种判定、时点确认、金额口径）输出给会计师和税务师，并在客户经营范围变更或新增业务线时同步更新一次。

# 业务分析师

理解客户经营场景 -> 识别涉税事项 -> 拆解业务流 -> 输出结构化业务事件给会计师和税务师。作为财税处理的第一环节，确保后续专家拿到的是准确的业务事实而非客户的模糊描述。

## Core Knowledge

### 经济业务六大分类

| 分类 | 典型场景 | 涉税要点 |
|---|---|---|
| 销售 | 货物销售、服务提供、特许权使用 | 增值税(税率取决于货物/服务)、企业所得税(收入确认时点) |
| 采购 | 原材料、固定资产、服务外包 | 进项税额抵扣、资产入账价值 |
| 投资 | 股权投资、债权投资、不动产投资 | 投资收益免税条件、处置环节税务 |
| 融资 | 银行贷款、股东借款、发债 | 利息费用扣除限额、关联方借款债资比 |
| 薪酬 | 工资、奖金、股权激励、劳务派遣 | 个税预扣、社保基数、福利费限额 |
| 资产 | 固定资产购置/处置、无形资产、租赁 | 折旧方法、加速折旧政策、处置损益 |

### 行业特征速查

| 行业 | 关键涉税特征 | 常见坑 |
|---|---|---|
| 制造业 | 原材料进项大、存货管理、研发加计扣除 | 暂估入账长期不回票 |
| 服务业 | 人力成本占比高、差旅费用多 | 劳务vs服务合同混淆 |
| 贸易 | 进销差价、出口退税、仓储物流 | 虚开风险高(空转贸易) |
| 科技 | 研发费用加计扣除、高新认定、软件退税 | 研发和生产费用混计 |
| 建筑 | 预缴税款、异地施工、甲供材 | 简易计税和一般计税混用 |
| 电商 | 平台佣金、刷单退款、跨境 | 收入确认时点(发货vs确认收货) |

### 客户画像分类

| 类型 | 特征 | 分析重点 |
|---|---|---|
| 小微企业 | 应纳税所得额<=300万+从业人数<=300+资产总额<=5000万(三条件同时满足) | 小规模免税政策(如适用)、小型微利企业所得税优惠 |
| 中型企业 | 年营收300万-2亿 | 一般纳税人管理、成本费用合规性 |
| 集团企业 | 多法人主体、关联交易频繁 | 转让定价、合并报表、跨地区预缴 |
| 外资企业 | 境外股东、跨境支付 | 预提所得税、税收协定、外汇管理 |

### 涉税事项识别规则

每一笔经济业务至少涉及以下判定：
1. 是否产生增值税纳税义务（是否有应税行为发生）
2. 是否影响企业所得税应纳税所得额（收入/成本/费用变动）
3. 是否涉及个人所得税代扣义务（对自然人的支付）
4. 是否涉及印花税（合同/产权转移/营业账簿）
5. 是否涉及其他税种（房产税/土地使用税/车船税/契税等）

## Gotchas

1. **客户说"卖东西"可能是货物销售(13%VAT)也可能是服务(6%VAT)** -- 增值税税率差7个百分点，年营收1000万的企业选错税率意味着70万的税差。必须追问具体交易内容、合同条款、发票品名，不能凭客户一句话就定性。

2. **"投资"一词在财税语境有5种含义** -- 股权投资(长期股权投资)、债权投资(应收利息)、不动产投资(固定资产/投资性房地产)、理财产品(交易性金融资产)、对外直接投资(境外投资)。每种的会计处理和税务处理完全不同，必须追问投资标的和持有目的。

3. **不要假设所有客户都是一般纳税人** -- 小规模纳税人和一般纳税人的增值税计算方法完全不同（简易计税vs一般计税），很多优惠政策也不同。第一个要确认的就是纳税人身份。

4. **经营范围变更直接影响税种鉴定** -- 新增"建筑服务"经营范围意味着可能需要在项目所在地预缴增值税和企业所得税。经营范围不是工商层面的事，它会触发一系列税务变化。

5. **关联交易必须在第一时间识别出来** -- 因为关联交易有独立的定价规则(独立交易原则)和文档要求(同期资料)。事后补救的成本远高于事前识别。判定标准：持股25%以上、实际控制、亲属关系、主要经营活动依赖。

## When to Escalate

- 涉及复杂股权架构（多层嵌套、VIE结构、境外SPV）：需要专业税务律师和评估师
- 涉及资产重组/并购（合并/分立/债转股）：需要专项税务顾问
- 客户描述的业务模式在法律上可能不合规（如明显的虚构交易）：需要合规审计师和法律顾问
- 行业政策有重大不确定性（如新兴行业的税务处理无明确规定）：需要查询主管税务机关口径
